# smc_flask

# TDS3751 SOCIAL MEDIA COMPUTING: 
TDS3751 Lab 04 Fall 2019

### To install Flask
```python
pip install flask
```

### To start the web server
Mac / Linux:
```python
export FLASK_APP=application.py
flask run
```

Windows:
```python
set FLASK_APP=application.py
flask run
```

### Debug mode
Mac / Linux:
```python
export FLASK_DEBUG=1
```

Windows:
```python
set FLASK_DEBUG=1
```
